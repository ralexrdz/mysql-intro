¿Qué es una Base de Datos?

Es un conjunto de datos , pertenecientes a un mismo contexto , organizados para sea accesible y gestionada fácilmente.

> «Un dato aislado sin un contexto es irrelevante para la toma de decisiones.»

Un Sistema de base de datos consiste en una colección de:

  • Datos
  • DBMS - Sistema de Gestion de Base de datos
  • Lenguaje estructurado de acceso
  • Procedimientos, índices, vistas, escuchadores
  • Herramientas de reporte

Tipos de bases de datos

Cuando nos referimos al tipo de base de datos estamos hablando de su modelo de datos es decir, la estructura lógica utilizada para construir esa base de datos

  • Relacionales
  • No Relacionales
  • Orientada a Objetos
  • Orientadas a Grafos
  • Llave-Valor
  • Wide-column stores

¿Qué es una base de datos Relacional ?

  • Almacena y gestiona registros de datos relacionados entre sí
  • Representa los datos en tablas. Donde cada fila es un registro y cada columna un tipo de dato
  • Los tipos de campos y las relaciones entre tablas son predefinidas
  • Planteab una serie de reglas de integridad de los datos para garantizar que sea esta base de datos sea accesible, escalable, rápida y precisa

Más conocidas: Microsoft SQL Server, Oracle Database, MySQL e IBM DB2

Cómo se representa una base de datos relacional

Existen varias maneras, pero destacan dos modelos:

  • Modelo Entidad Relacion
  • Modelo Relacional

Structured Query Language (SQL)

Lenguaje estándar y más utilizado en bases de datos relacionales

Se divide en 3 tipos de lenguajes

  • DDL Data Definition Language
  • DML Data Manioulation Language
  • DCL Data Control Language

Instalación de MySQL

Utilizaremos docker para levantar una imagen de docker

Validamos la instalación de docker abriendo nuestra terminal y corriendo el siguiente comando


              docker -v

Descarga de la imagen


            docker pull mysql

Puedes fácilmente instalar la versión que gustes


            docker pull mysql:<tag>

Hub de la imagen de mysql

Correr un contenedor de mysql 


            docker run --name sodimac-mysql 
            -e MYSQL_ROOT_PASSWORD=my-secret-pw -d mysql

...o para abrir el puerto del contenidor a un puerto local


            docker run 
            --name sodimac-mysql -p 3306:3306 
            -e MYSQL_ROOT_PASSWORD=my-secret-pw -d mysql

Entrar al contenedor a probar comandos de mysql


            docker exec --it sodimac-mysql bash

dentro, debería de verse algo así


            root@2aaaa0f6d1e7:/#

correr comando: de igreso a CLI de mysql


            mysql -uroot -p
